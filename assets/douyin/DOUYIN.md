# 抖音素材存储规则

> 目录：`assets/douyin/`  
> 最后更新：2026-05-15

## 1. 目录定位

`douyin/` 用于存放只服务于抖音平台发布的素材成品与平台适配文件。

判断标准：如果素材已经按照抖音短视频、封面、字幕、竖屏比例或发布需求做了适配，就放在这里。

## 2. 适合存放

- 抖音竖屏成片、剪辑工程导出版本
- 抖音视频封面、字幕版视频、口播切片
- 抖音平台专用贴纸、字幕条、片头片尾
- 抖音账号主页截图、内容参考截图
- 从通用素材或 campaign 素材二次适配后的抖音版本

## 3. 不适合存放

- 未经适配的拍摄原片、通用场景照片
- 品牌 Logo、VI规范、统一头像等品牌基础资产
- 微信或小红书专用成品
- 抖音后台导出的运营数据

## 4. 建议子目录

```text
douyin/
├── covers/       # 抖音视频封面、合集封面、封面源文件
├── videos/       # 抖音竖屏成片、剪辑导出、口播切片、字幕版视频
├── subtitles/    # 字幕文件、字幕条素材、口播文稿对应字幕
├── screenshots/  # 账号主页截图、视频截图、评论截图、参考案例截图
├── templates/    # 抖音封面模板、字幕模板、片头片尾模板
└── archive/      # 旧版成片、废弃封面、历史发布素材归档
```

## 5. 命名建议

- 视频成片：`YYYY-MM-DD_账号_视频主题_抖音成片.扩展名`
- 视频封面：`YYYY-MM-DD_账号_视频主题_封面.扩展名`
- 字幕文件：`YYYY-MM-DD_账号_视频主题_字幕.扩展名`

## 6. 使用原则

- 抖音发布成品可以从 `brand/`、`shared/`、`campaigns/` 复制素材后再适配。
- 不在本目录存放后台原始数据；数据应进入对应账号的 `data_raw/`。
- 已存在的素材目录不要随意改名或删除；整理前先确认用途。
