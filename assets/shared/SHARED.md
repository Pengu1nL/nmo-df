# 通用素材存储规则

> 目录：`assets/shared/`  
> 最后更新：2026-05-15

## 1. 目录定位

`shared/` 用于存放不绑定某个平台、某个账号或某次活动，但可以被多类内容反复使用的通用素材。

判断标准：如果素材以后可能被多个平台、多篇内容或多个项目复用，但它不是品牌规范资产，也不是活动专属素材，就放在这里。

## 2. 适合存放

- 办公空间、门店、场地、活动现场、服务场景等环境素材
- 团队日常、工作流程、服务过程、客户沟通等通用素材
- 人物工作照、团队协作、咨询沟通、产品或服务展示素材
- B-roll 视频、空镜、环境镜头、转场镜头、补充画面
- 通用背景图、通用贴纸、通用配乐、通用转场素材
- 可长期复用但未绑定具体 campaign 的素材包

## 3. 不适合存放

- Logo、VI规范、品牌色、账号头像等品牌基础资产
- 某次线下活动、营销节点、专题推广的完整活动素材
- 已经专门适配某个平台尺寸和发布形式的成品
- 原始运营数据、后台导出文件

## 4. 建议子目录

```text
shared/
├── locations/   # 办公空间、门店、场地、公共空间、环境空镜
├── scenes/      # 工作流程、服务过程、活动现场、使用场景素材
├── people/      # 团队、用户、客户、嘉宾等人物素材
├── team/        # 团队工作照、协作场景、人物介绍素材
├── b-roll/      # 通用视频空镜、环境镜头、转场镜头、补充画面
├── music/       # 通用背景音乐、音效、环境声素材及授权说明
└── templates/   # 通用版式模板、可跨平台复用的设计模板
```

## 5. 命名建议

- 图片：`YYYY-MM-DD_场景_简短说明_序号.扩展名`
- 视频：`YYYY-MM-DD_场景_镜头类型_序号.扩展名`
- 素材包：`YYYY-MM-DD_主题_素材包`

## 6. 使用原则

- 涉及人物正脸、姓名、联系方式、用户评价、案例数据、作品或成果展示的素材，应确认授权或标注使用限制。
- 不确定授权状态的素材，不要直接用于对外发布。
- 已确认高频复用的素材，可以在 Notion 素材索引库中登记路径与授权状态。
