# B站素材存储规则

> 目录：`assets/bilibili/`  
> 最后更新：2026-05-15

## 1. 目录定位

`bilibili/` 用于存放只服务于 B站 / bilibili 平台发布的素材成品与平台适配文件。

判断标准：如果素材已经按照 B站视频、封面、字幕、动态、专栏或合集发布需求做了尺寸、标题、版式、字幕或导出适配，就放在这里。

## 2. 适合存放

- B站横屏或竖屏成片、剪辑工程导出版本
- B站视频封面、合集封面、分 P 封面
- 字幕文件、口播稿对应字幕、弹幕或评论截图
- B站动态配图、专栏头图、专栏正文配图
- B站账号主页截图、内容参考截图、竞品案例截图
- 从通用素材或 campaign 素材二次适配后的 B站版本

## 3. 不适合存放

- 未经适配的拍摄原片、通用场景照片
- 品牌 Logo、VI规范、统一头像等品牌基础资产
- 微信、小红书或抖音专用成品
- B站后台导出的运营数据

## 4. 建议子目录

```text
bilibili/
├── covers/         # B站视频封面、合集封面、封面源文件
├── videos/         # B站成片、剪辑导出、分 P 视频、字幕版视频
├── subtitles/      # 字幕文件、字幕条素材、口播文稿对应字幕
├── dynamic-posts/  # B站动态配图、动态发布素材、互动运营图片
├── columns/        # B站专栏头图、正文配图、长图切片
├── screenshots/    # 主页截图、视频截图、评论截图、弹幕截图、参考案例截图
├── templates/      # B站封面模板、字幕模板、片头片尾模板
└── archive/        # 旧版成片、废弃封面、历史发布素材归档
```

## 5. 命名建议

- 视频成片：`YYYY-MM-DD_账号_视频主题_B站成片.扩展名`
- 视频封面：`YYYY-MM-DD_账号_视频主题_B站封面.扩展名`
- 字幕文件：`YYYY-MM-DD_账号_视频主题_字幕.扩展名`
- 动态配图：`YYYY-MM-DD_账号_动态主题_配图_序号.扩展名`
- 专栏配图：`YYYY-MM-DD_账号_专栏主题_正文图_序号.扩展名`

## 6. 使用原则

- B站发布成品可以从 `brand/`、`shared/`、`campaigns/` 复制素材后再适配。
- 不在本目录存放后台原始数据；数据应进入对应账号的 `data_raw/`。
- 涉及用户评论、弹幕、头像、昵称、投稿截图或案例数据时，应确认授权或做脱敏处理。
- 废弃但可能追溯的旧版成品放入 `archive/`。
