# 微信素材存储规则

> 目录：`assets/wechat/`  
> 最后更新：2026-05-15

## 1. 目录定位

`wechat/` 用于存放只服务于微信平台发布或排版的素材成品与平台适配文件。

判断标准：如果素材已经按照微信公众号、视频号或微信图文排版需求做了尺寸、格式、封面或排版适配，就放在这里。

## 2. 适合存放

- 微信公众号头图、封面图、正文配图
- 微信长图、推文排版图片、二维码引导图
- 视频号封面、视频号发布成品
- 微信专用海报、菜单图、自动回复配图
- 从通用素材或 campaign 素材二次适配后的微信版本

## 3. 不适合存放

- 未经平台适配的原始拍摄素材
- 品牌 Logo、VI规范、统一头像等品牌基础资产
- 小红书或抖音专用封面和发布成品
- 微信后台导出的运营数据

## 4. 建议子目录

```text
wechat/
├── covers/          # 公众号头图、推文封面、视频号封面
├── article-images/  # 推文正文配图、长图切片、排版用图片
├── posters/         # 微信专用海报、朋友圈图、社群转发图
├── video-account/   # 视频号成片、视频号封面、视频号发布素材
├── qrcode/          # 公众号二维码、咨询二维码、活动报名二维码
└── archive/         # 旧版封面、废弃设计、历史推文素材归档
```

## 5. 命名建议

- 公众号封面：`YYYY-MM-DD_账号_推文主题_封面.扩展名`
- 正文配图：`YYYY-MM-DD_账号_推文主题_正文图_序号.扩展名`
- 视频号封面：`YYYY-MM-DD_账号_视频主题_视频号封面.扩展名`

## 6. 使用原则

- 微信平台成品可以从 `brand/`、`shared/`、`campaigns/` 复制素材后再适配。
- 不在本目录存放后台原始数据；数据应进入对应账号的 `data_raw/`。
- 废弃但可能追溯的旧版成品放入 `archive/`。
